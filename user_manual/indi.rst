================================================
Chapter 9. Astronomical Device Control with INDI
================================================

         KStars provides an interface to configure and control
         astronomical instruments via the
         :ref:`INDI <indi-faq-what-is-indi>` protocol.

         The INDI protocol supports a variety of astronomical
         instruments such as CCD cameras and focusers. For an up to date
         list of supported devices, please visit INDI `supported devices
         page <https://indilib.org/index.php?title=Devices>`__.
